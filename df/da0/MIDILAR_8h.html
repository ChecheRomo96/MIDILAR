Detailed Description

Main include file for the MIDILAR library.

This header file serves as the central point for including all necessary headers for using the MIDILAR library. It conditionally includes additional modules based on the build settings.

  • Includes build settings defined in MIDILAR_BuildSettings.h.
  • Optionally includes system tools if MIDILAR_SYSTEM_TOOLS_ENABLED is defined.
